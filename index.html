About

I'm Alice, a 25 years old Front-end developer based in Brighton, UK.

This portfolio represents mainly the projects I am working on while studying a Front-End Developer Nanodegree at Udacity.

I am learning to master the skills required to become an awesome Front-End Web Developer, and start building beautiful, responsive websites optimized for mobile and desktop performance.

Featured Projects

Build My Portfolio Site

This is my first project created for the Udacity Front-End Development Nanodegree program and it helped me to better understand responsive design. In this project, the goal was to replicate a design mockup in HTML and CSS. The website had to be responsive and display images, descriptions and links to my projects. I have used Bootstrap framework to build my portfolio and I have spent a lot of time exploring what it has to offer.

I have also discovered version control tools such as Git and Github while working on this project.
