About

I'm Alice, 26 years old and living in London, UK.

In 2016, I decided to start a life-long learning journey to master the skills required to become an awesome Full-Stack Developer.

This portfolio represents the projects I have been working on while learning more about Front-end development as well as Back-end development.

Featured Projects

Build My Portfolio Site

This is my first project created for the Udacity Front-End Development Nanodegree program and it helped me to better understand responsive design. In this project, the goal was to replicate a design mockup in HTML and CSS. The website had to be responsive and display images, descriptions and links to my projects. I have used Bootstrap framework to build my portfolio and I have spent a lot of time exploring what it has to offer.

I have also discovered version control tools such as Git and Github while working on this project.

Check it out on Github

Interactive Online Resume

As part of the Udacity Front-End Development Nanodegree program, I have built an online resume using HTML5, CSS3, JavaScript, and jQuery.

In this project, the goal was to transform a static web pages into dynamic applications. I have used variables, data structures, conditional statements, loops, and function in JavaScript as well as the core features of jQuery to manipulate the DOM.

I have also made sure that the page is fully responsive.

Check it out on Github

Classic Arcade Game

I was provided with visual assets and a game loop engine; using these tools I have added a number of entities to the game including the player characters and enemies to recreate a classic arcade game.

I learnt JavaScript’s object oriented programming features to write classes capable of creating countless instances of similarly functioning objects. I have discovered a variety of ways inheritance and delegation used to create well architected and performant applications.

Check it out on Github

Website optimisation

Provided with a website with a number of optimization- and performance-related issues I improved it so that it achieves a PageSpeed score of 90 and runs at 60 frames per second.

I've learnt about the critical rendering path, the process by which the browser receives HTML, CSS and JavaScript and the required processing to turn them into rendered pixels.

Check it out on Github

Your next stop for a coffee in London

This is a single-page application that I have created featuring a map of the 10 best coffee shops in London.

I added some functionnalities to this map including highlighted locations, third-party data about those locations using third-party APIs (Google Maps and Foursquare) and various ways to browse the content.

I've also learned how to use MV* frameworks such as KnockoutJS that I have used for this project.

Check it out on Github

Feed reader testing

For this project I was given a web-based application that reads RSS feeds.

I learnt how to use Jasmine to write a number of tests against a pre-existing application to test the underlying business logic of the application as well as the event handling and DOM manipulation.

Check it out on Github
